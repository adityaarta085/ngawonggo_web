# Portal Resmi Desa Ngawonggo

Portal informasi digital terintegrasi untuk Desa Ngawonggo, Kecamatan Kaliangkrik, Kabupaten Magelang.

## Fitur Utama
- **Landing Page Dinamis:** Menampilkan Berita Terbaru dan Statistik Desa.
- **Profil 10 Dusun:** Halaman detail untuk setiap dusun dengan statistik dan informasi masjid.
- **Layanan Publik:** Sistem pengaduan online dan daftar instansi terkait (Pemerintah & Keamanan).
- **Digital Quran:** Fitur murottal Al-Quran dengan kemampuan resume audio dan pembaruan rentang otomatis.
- **Admin Panel:** Manajemen konten penuh untuk berita, dusun, instansi, dan statistik.

## Quick Setup Supabase

1. Buat project baru di [Supabase](https://supabase.com/).
2. Jalankan SQL berikut di SQL Editor Supabase untuk menyiapkan skema database.
3. Konfigurasikan variabel lingkungan (.env):
   ```
   REACT_APP_SUPABASE_URL=your_url
   REACT_APP_SUPABASE_ANON_KEY=your_key
   ```

### Skema Database (SQL)

```sql

-- 1. TABEL BERITA

CREATE TABLE news (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT NOT NULL,
  content TEXT,
  image TEXT,
  video_url TEXT,
  date TEXT,
  category TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 2. TABEL DUSUN

CREATE TABLE dusuns (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  description TEXT,
  population TEXT,
  area TEXT,
  houses TEXT,
  map_link TEXT,
  image_url TEXT,
  masjid_name TEXT,
  masjid_image TEXT,
  masjid_location_url TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 3. TABEL INSTANSI & LAYANAN

CREATE TABLE institutions (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT NOT NULL,
  image TEXT,
  category TEXT, -- 'pemerintah', 'layanan', 'keamanan'
  website_url TEXT,
  location_url TEXT,
  address TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 4. TABEL STATISTIK DESA

CREATE TABLE village_stats (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  label TEXT NOT NULL,
  value TEXT,
  icon TEXT,
  color TEXT
);

-- 5. TABEL PENGADUAN & PESAN

CREATE TABLE complaints (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  contact TEXT,
  category TEXT,
  status TEXT DEFAULT 'open',
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE complaint_messages (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  complaint_id TEXT REFERENCES complaints(id) ON DELETE CASCADE,
  sender_type TEXT NOT NULL,
  message TEXT,
  image_url TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 6. TABEL SISTEM LAINNYA

CREATE TABLE announcements (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  content TEXT NOT NULL,
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE comments (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  news_id BIGINT REFERENCES news(id) ON DELETE CASCADE,
  parent_id BIGINT REFERENCES comments(id) ON DELETE CASCADE,
  name TEXT NOT NULL,
  email TEXT,
  content TEXT NOT NULL,
  is_approved BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE popups (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT,
  content TEXT,
  type TEXT DEFAULT 'text',
  button_label TEXT,
  button_link TEXT,
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE site_settings (
  key TEXT PRIMARY KEY,
  value TEXT
);
```

I have implemented the Admin Broadcast Email feature for the Desa Ngawonggo website.
Key changes:
1. Installed 'nodemailer' for email handling.
2. Created 'api/broadcast.js' (Vercel/Node.js API) to handle SMTP logic with Gmail. It uses the provided App Password.
3. Created 'get_all_users()' RPC function in Supabase to fetch emails from 'auth.users'.
4. Created 'BroadcastManager.js' component in the admin dashboard:
   - Form with Subject and React Quill editor.
   - Recipient selection: All users vs Selected users.
   - Searchable and multi-select user list for "Selected Users".
   - Batch sending logic: sends 2 emails, waits 2 seconds, then continues.
   - Progress indicator showing successful and failed counts.
   - "Send test email to myself" option.
5. Integrated 'BroadcastManager' into the 'AdminPage' sidebar and main content.
